Lightbox Plugin

It is a jQuery plugin and it is open source. Below you can find the features and sample.

Here are its features

  1. Image size is auto adjustable
  2. you can give title for the images
  3. there are some keyboard shortcuts for next and previous images

  • keyboard shortcuts are p-prev,n-next,c-close
  • you can change the overlay color,opacity,shortcut keys in js file
  • the size of the title can be changed in css
